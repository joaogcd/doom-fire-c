# Doom Fire in C using SDL3

This is a implementation of the Doom Fire effect in C using SDL3.

![](images/doom_fire.gif)

## Building
To build the project, run the following command in the terminal:

```bash
make
``` 
This will compile the source code and create an executable in the `bin` directory.

## Running
To run the project, use the following command:

```bash
make run
```
This will execute the compiled program and display the Doom Fire effect in a window.
