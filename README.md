# Doom Fire in C using SDL3 🔥

This is an implementation of the Doom Fire effect in C using SDL3.

![](images/doom_fire.gif)

The algorithm is based on manipulating a 2D array of color values that represent the fire. The fire is generated by randomly igniting the bottom row of the array and then propagating the fire upwards by averaging the color values of the neighboring pixels.

## Building
To build the project, run the following command in the terminal:

```bash
make
``` 
This will compile the source code and create an executable in the `bin` directory.

## Running
To run the project, use the following command:

```bash
make run
```
This will execute the compiled program and display the Doom Fire effect in a window.
